##元素

###rect

> 绘制方形

| 属性         | 取值   | 作用     |
| ------------ | ------ | -------- |
| x/y          | number |          |
| rx/ry        | number | 圆角     |
| width/height | number |          |
| stroke       | color  | 边框色   |
| fill         | color  | 填充色   |
| stroke-width | number | 边框宽度 |



### circle

> 绘制圆形

| 属性         | 取值   | 作用     |
| ------------ | ------ | -------- |
| cx/cy        | number |          |
| r            | number | 半径     |
| stroke       | color  | 边框色   |
| fill         | color  | 填充色   |
| stroke-width | number | 边框宽度 |

###ellipse

> 绘制椭圆

| 属性         | 取值   | 作用        |
| ------------ | ------ | ----------- |
| cx/cy        | number |             |
| rx/ry        | number | x/y轴上半径 |
| stroke       | color  | 边框色      |
| fill         | color  | 填充色      |
| stroke-width | number | 边框宽度    |



###line

> 绘制直线

| 属性   | 取值   | 作用         |
| ------ | ------ | ------------ |
| x1/x2  | number | 起/终点x坐标 |
| y1/y2  | number | 起/终点y坐标 |
| stroke | color  | 线条色       |



###polyline

> 绘制多段折线

| 属性         | 取值                  | 作用     |
| ------------ | --------------------- | -------- |
| point        | x1 y1 x2 y2 x3 y3.... | 折点坐标 |
| stroke       | color                 | 线条色   |
| fill         | color                 | 填充色   |
| stroke-width | number                | 边框宽度 |



###polygon

> 绘制多段折线（首尾自动闭合）

| 属性         | 取值                  | 作用     |
| ------------ | --------------------- | -------- |

| point        | color                 | 线条色   |
| fill         | color                 | 填充色   |
| stroke-width | number                |          |
