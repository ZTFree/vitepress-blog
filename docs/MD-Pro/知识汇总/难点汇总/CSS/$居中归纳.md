# 居中归纳

| 名称                 | 相关属性            | 简述                   |
| -------------------- | ------------------- | ---------------------- |
| 文本/行内块水平居中  | text-align          | 常用                   |
| 文本垂直居中         | line-height         | 常用                   |
| 盒子水平居中：外边距 | margin              | 常用                   |
| 行内块垂直居中       | vertical-align      | 又称中线居中           |
| 背景图片居中         | background-position | 用于背景图片的居中     |
| 计算属性定位居中     | positon，calc（）   | 用于已知盒子宽高的居中 |
| 强制性定位居中       | position，margin    | 用于未知盒子宽高的居中 |
| flex居中             |                     |                        |
