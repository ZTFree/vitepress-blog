# 普通函数与箭头函数对比

|                           | 普通函数       | 箭头函数                 |
| ------------------------- | -------------- | ------------------------ |
| this指向                  | 取决于调用方式 | 与定义上下文this指向一致 |
| arguments局部遍历（对象） | 有             | 没有                     |
| 能否作为构造函数          | 能             | 不能                     |
| prototype属性             | 有             | 无                       |
