# innerHTML、innerText、textContent三者的区别

| 区别                   | innerHTML | innerText      | textContent                |
| ---------------------- | --------- | -------------- | -------------------------- |
| 文本内容               | 保留      | 保留           | 保留                       |
| 标签                   | 保留      | 不保留         | 不保留                     |
| 行头缩进/空白          | 保留      | 不保留         | 保留                       |
| 注释                   | 保留      | 不保留         | 不保留（注释行会保留空行） |
| 赋值为html标签解析结果 | html结构  | 字符转义字符串 | 字符转义字符串             |
